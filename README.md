# ft-images-fedex

A friendly and intuitive interface for viewing and navigating family photos from large scanning projects, with photos grouped by person using face recognition.

## Project Overview

After a large scanning project, people are often given a hard drive full of photos that are difficult to navigate and consume. This tool solves that by providing a clean interface to view all family photos, automatically grouped by the people in them.
